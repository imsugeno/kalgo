# 制御フロー

## 学習テーマ
- `if`/`else` 条件分岐（式としても使用可能）
- `when` 式（パターンマッチング）
- `for` ループとレンジ（`..`, `..<`, `downTo`, `step`）
- `while`/`do-while` ループ

## 問題

標準入力から整数 N を読み取り、1 から N までの各数字に対して FizzBuzz を行ってください。

### ルール
- 15の倍数 → `FizzBuzz`
- 3の倍数（15の倍数を除く） → `Fizz`
- 5の倍数（15の倍数を除く） → `Buzz`
- それ以外 → その数字をそのまま出力

**`when` 式を使って実装してください。**

### 入力形式
```
N
```

### 出力形式
各数字の結果を1行ずつ出力

### 入出力例

**入力:**
```
20
```

**出力:**
```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
19
Buzz
```

## ヒント
- `for (i in 1..N)` で1からNまでループできます
- `when` 式の条件では `i % 15 == 0` のようにブーリアン式が使えます
- `when` の分岐は上から順に評価されます（最初にマッチした分岐が実行される）
