## 問題

時間を秒に変換するプログラムを作成してください。

標準入力から変換クエリの数 N を読み取り、続いて N 行のクエリを処理します。
各クエリは `hours minutes seconds` の形式です。
これらを合計秒数に変換して出力してください。

**`intervalInSeconds` 関数を作成し、デフォルト引数を活用してください。**

### 入力形式
```
N
hours1 minutes1 seconds1
hours2 minutes2 seconds2
...
```

### 出力形式
各クエリの合計秒数を1行ずつ出力

### 入出力例

**入力:**
```
4
1 20 15
0 1 25
2 0 0
0 10 0
```

**出力:**
```
4815
85
7200
600
```

## ヒント
- 関数のデフォルト引数: `fun f(x: Int = 0)`
- 1時間 = 3600秒、1分 = 60秒
- 単一式関数: `fun f(x: Int): Int = x * 2`
