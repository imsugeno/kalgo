# 基本型と算術演算

## 学習テーマ
- 基本型（`Int`, `Double`, `Boolean`, `String`）
- 算術演算子（`+`, `-`, `*`, `/`, `%`）
- 型変換（`toInt()`, `toDouble()` など）

## 問題

標準入力から2つの整数 `a`, `b` を読み取り、以下の算術演算の結果を出力してください。
さらに、`a` を `Double` に変換して `b` で割った結果も出力してください。

### 入力形式
```
a
b
```

### 出力形式
```
a + b = [結果]
a - b = [結果]
a * b = [結果]
a / b = [結果（整数除算）]
a % b = [結果]
a.toDouble() / b = [結果（小数点以下含む）]
```

### 入出力例

**入力:**
```
17
5
```

**出力:**
```
a + b = 22
a - b = 12
a * b = 85
a / b = 3
a % b = 2
a.toDouble() / b = 3.4
```

## ヒント
- Int 同士の除算は整数除算になります（小数点以下切り捨て）
- `toDouble()` で Int を Double に変換できます
- 文字列テンプレートを活用しましょう
